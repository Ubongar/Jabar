# Jabar

Student Management System built with Spring Boot.

## Prerequisites

- Java 21
- Maven 3.9+
- MySQL 8+

## Environment Variables

Set these in your shell before running:

- `DB_URL` (default: `jdbc:mysql://127.0.0.1:3306/student_db`)
- `DB_USERNAME` (default: `root`)
- `DB_PASSWORD` (default: empty)

## Run

```powershell
mvn clean spring-boot:run
```
